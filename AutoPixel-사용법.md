# AutoPixel — wplace 자동 픽셀 (AutoHotkey) · v1.1.0

실제 마우스/키보드를 자동 조작해 **수동 동작을 그대로 재현**하는 오토마우스입니다.
크롬 확장(`wplace-2click`)과는 **완전히 별개의 프로그램**이며, 이 폴더 하나로 자기완결적으로 동작합니다. (폴더째 어디로 옮겨도 됩니다.)

칸마다 순서: **이동 → `i`(색상 인식) → 클릭 0.1초 → 대기 0.1초 → 클릭 0.1초**

**두 가지 모드** (패널 상단의 둥근 버튼으로 선택)
- **직선 모드**: 시작(P1) → 끝(P2)을 N칸으로 균등 분할해 한 줄을 채움.
- **사각형 모드**: P1(좌상단) → P2(같은 행 끝) → P3(다른 행 끝) 3점으로, 첫 행부터 마지막 행까지 행 단위로 채움.

> 패널의 **키캡**은 해당 키를 누르면 초록으로 하이라이트되고, **F4(실행)** 는 작업이 끝날 때까지 켜져 있어 "어떤 키/작업 중인지"를 한눈에 볼 수 있습니다.

---

## 1. 준비

1. **AutoHotkey v2** 설치 → https://www.autohotkey.com (반드시 **v2**)
2. 이 폴더의 **`AutoPixel-실행.bat`** 더블클릭 → 좌측 상단에 패널이 뜹니다.
   - 잘 안 되면 **`AutoPixel-진단.bat`** 을 실행해 원인을 확인하세요.
   - 종료: 트레이의 초록 `H` 아이콘 우클릭 → Exit / 수정 후 Reload

## 2. 단축키

| 키 | 기능 |
|---|---|
| **F2** | **P1 시작점** 보정 (사각형: 좌상단 칸) |
| **F3** | **P2 끝점** 보정 — 직선: 끝 칸 / 사각형: **같은 행의 끝 칸**(우상단) |
| **F7** | **P3 다른 행 끝** 보정 — **사각형 모드 전용**(마지막 행의 칸) |
| **F8** | **1칸 크기 보정** — 인접한 두 칸을 차례로 찍으면 칸 크기를 학습 → **칸 수 자동 계산** |
| **F4** | **실행** |
| **Esc** 또는 **F6** | **정지** (실행 중에만 동작) |

> 패널 상단에 같은 키들이 **키캡(3열 × 2행)** 으로 배치되어 있고, 누르면 하이라이트됩니다.

## 3. 패널 구성

- **모드 버튼**(둥근): `직선` / `사각형` — 클릭으로 전환 (현재 모드는 초록).
- **키캡 행**: `F2 · F3 · F7 · F8 · F4 · Esc` — 누르면 하이라이트.
- **좌표 표시**: P1 / P2 / P3 — 보정되면 `●` 초록으로 바뀜.
- **1칸 크기**: F8 보정 결과(px) 표시.
- **설정값**
  - **칸 수 N** — 직선: 시작~끝 포함 칸 개수 / 사각형: 한 행의 가로 칸 수 (**보정 시 자동 입력**)
  - **딜레이 (ms)** — 칸 사이 간격 (기본 **50**)
  - **지터 (%)** — 딜레이를 ± 무작위로 흔드는 비율, 규칙적 패턴 완화 (기본 **25**)

## 4. 1칸 크기 보정 → 칸 수 자동 (권장)

큰 그림에서 칸을 일일이 세지 않으려면 **딱 한 번** 1칸 크기를 보정하세요.

1. 임의의 한 칸 중앙에 마우스 올리고 **F8**.
2. **바로 옆 칸** 중앙에서 **F8** 한 번 더. → `1칸 크기: NNpx (보정됨)` 표시.
3. 이후 **F2/F3로 시작·끝을 찍으면 `칸 수 N`이 자동으로 채워집니다.** (필요하면 직접 수정 가능)

> **줌(확대/축소)을 바꾸면 칸 크기가 달라지므로 F8을 다시** 해주세요.
> 보정하지 않아도 `칸 수 N`을 직접 입력해 그대로 사용할 수 있습니다.

## 5. 직선 모드 사용법

1. **직선** 버튼 선택.
2. (권장) **F8**로 1칸 크기 보정.
3. 첫 칸 중앙에서 **F2**, 마지막 칸 중앙에서 **F3** → `칸 수 N` 자동 입력 (미보정 시 직접 입력).
4. **F4** 실행 / **Esc** 정지.

## 6. 사각형 모드 사용법

1. **사각형** 버튼 선택.
2. (권장) **F8**로 1칸 크기 보정.
3. **P1**: 좌상단(시작) 칸 중앙에서 **F2**.
4. **P2**: P1과 **같은 행의 마지막 칸**(우상단) 중앙에서 **F3** → 가로 `칸 수 N` 자동 입력.
5. **P3**: **마지막 행**의 칸에서 **F7** (그 행 높이만 맞으면 가로 위치는 무시).
6. **F4** 실행 → 첫 행부터 마지막 행까지 **왼→오, 위→아래 순서**로 채워집니다.

> **세로 행 수는 자동 계산됩니다.** wplace 픽셀은 정사각형이라, 1칸 크기로 P3까지의 세로 거리를 나눠 행 수를 정합니다.
> 처음엔 작은 사각형(예: 가로 N=3, P3를 2~3행 아래)으로 테스트해 정렬을 확인한 뒤 키우세요.

## 7. 주의사항

- **영문 입력 상태**에서 사용 (한글 IME가 켜져 있으면 `i`가 색상 인식으로 안 갈 수 있음).
- 실행 중에는 **마우스·키보드를 건드리지 마세요.**
- 실행 전 **wplace 창이 보이는 상태**여야 합니다. (F4 시 시작 좌표의 창을 자동 활성화)
- 가로/세로 격자만 그립니다. (점이 약간 어긋나도 주축으로 자동 정렬)
- 페인트 충전량을 소모합니다. 큰 사각형은 충전량을 빠르게 소진하니 N·행 수를 확인하세요.
- **줌을 바꾸면 F8(1칸 보정)과 F2·F3(·F7)을 다시** 찍으세요.
- 탐지는 보통 속도·규칙성으로 이뤄지므로 딜레이를 너무 짧게 하지 말고 지터를 적절히 두세요. 사용 책임은 사용자에게 있습니다.

## 9. 값 조정

`AutoPixel.ahk` 상단/내부의 값을 직접 수정할 수 있습니다.
- 클릭 타이밍: `PaintAt()`의 `Sleep 100`(클릭 0.1초) 등
- 색 인식↔클릭 간격: `Sleep 40`, `Sleep 60`
- 패널 색상: 상단 `C_GREEN`, `C_KEY` 등 색상 상수
