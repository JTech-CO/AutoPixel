# AutoPixel — wplace 자동 픽셀 (AutoHotkey)

실제 마우스/키보드를 자동 조작해 **수동 동작을 그대로 재현**하는 오토마우스입니다.
크롬 확장(`wplace-2click`)과는 **완전히 별개의 프로그램**이며, 이 폴더 하나로 자기완결적으로 동작합니다. (폴더째 어디로 옮겨도 됩니다.)

칸마다 순서: **이동 → `i`(색상 인식) → 클릭 0.1초 → 대기 0.1초 → 클릭 0.1초**

**두 가지 모드** (패널 상단에서 선택)
- **직선 모드**: 시작(P1) → 끝(P2)을 N칸으로 균등 분할해 한 줄을 채움.
- **사각형 모드**: P1(좌상단) → P2(같은 행 끝) → P3(다른 행 끝) 3점으로, 첫 행부터 마지막 행까지 행 단위로 채움.

---

## 1. 준비

1. **AutoHotkey v2** 설치 → https://www.autohotkey.com (반드시 **v2**)
2. 이 폴더의 **`AutoPixel-실행.bat`** 더블클릭 → 좌측 상단에 패널이 뜹니다.
   - 잘 안 되면 **`AutoPixel-진단.bat`** 을 실행해 원인을 확인하세요.
   - 종료: 트레이의 초록 `H` 아이콘 우클릭 → Exit / 수정 후 Reload

## 2. 단축키

| 키 | 기능 |
|---|---|
| **F2** | **P1 시작점** 보정 (사각형: 좌상단 칸) |
| **F3** | **P2 끝점** 보정 — 직선: 끝 칸 / 사각형: **시작과 같은 행의 끝 칸**(우상단) |
| **F7** | **P3 다른 행 끝** 보정 — **사각형 모드 전용**(마지막 행의 칸) |
| **F4** | **실행** |
| **Esc** 또는 **F6** | **정지** (실행 중에만 동작) |

## 3. 공통 설정값 (패널)

- **칸 수 N (가로)** — 직선: 시작~끝 포함 칸 개수 / 사각형: **한 행의 가로 칸 수**
- **딜레이 (ms)** — 칸 사이 간격
- **지터 (%)** — 딜레이를 ± 무작위로 흔드는 비율(규칙적 패턴 완화)

## 4. 직선 모드 사용법

1. 패널에서 **직선** 선택.
2. 첫 칸 중앙에 마우스 올리고 **F2**, 마지막 칸 중앙에서 **F3**.
3. **칸 수 N** = 시작~끝 포함 칸 개수 입력.
4. **F4** 실행 / **Esc** 정지.

## 5. 사각형 모드 사용법

1. 패널에서 **사각형** 선택.
2. **P1**: 좌상단(시작) 칸 중앙에 마우스 올리고 **F2**.
3. **P2**: **P1과 같은 행의 마지막 칸**(우상단) 중앙에서 **F3**.
4. **P3**: **마지막 행**의 칸(아무 칸이나 그 행 높이에 맞으면 됨)에서 **F7**.
5. **칸 수 N (가로)** = 한 행의 가로 칸 개수 입력.
6. **F4** 실행 → 첫 행부터 마지막 행까지 **왼→오, 위→아래 순서**로 채워집니다.

> **세로 행 수는 자동 계산됩니다.** wplace 픽셀은 정사각형이므로, P1→P2와 N으로 칸 간격을 구한 뒤
> 그 간격으로 P3까지의 세로 거리를 나눠 행 수를 정합니다. (P3는 **세로 위치(행)** 만 중요 — 가로 위치는 무시)
>
> 처음엔 작은 사각형(예: 가로 N=3, P3를 2~3행 아래)으로 테스트해 정렬을 확인한 뒤 키우세요.

## 6. 주의사항

- **영문 입력 상태**에서 사용 (한글 IME가 켜져 있으면 `i`가 색상 인식으로 안 갈 수 있음).
- 실행 중에는 **마우스·키보드를 건드리지 마세요.**
- 실행 전 **wplace 창이 보이는 상태**여야 합니다. (F4 시 시작 좌표의 창을 자동 활성화)
- 가로/세로 격자만 그립니다. (점이 약간 어긋나도 주축으로 자동 정렬)
- 페인트 충전량을 소모합니다. 큰 사각형은 충전량을 빠르게 소진하니 N·행 수를 확인하세요.
- 줌을 바꾸면 F2·F3(·F7)을 다시 찍으세요.
- 탐지는 보통 속도·규칙성으로 이뤄지므로 딜레이를 너무 짧게 하지 말고 지터를 적절히 두세요. 사용 책임은 사용자에게 있습니다.

## 8. 값 조정

`AutoPixel.ahk` 안의 `Sleep 100`(클릭 0.1초) 등을 직접 수정하면 타이밍을 바꿀 수 있습니다.
