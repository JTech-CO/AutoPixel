AutoPixel — Auto-pixel for wplace

Last Updated: 2026-06-14 · v1.0.0

AutoPixel is a Windows auto-mouse (an AutoHotkey script) that automatically draws pixels on wplace.live. This document explains how AutoPixel handles your information.

Summary — AutoPixel does not collect, store, or transmit any personal data, and runs entirely on your PC with no network communication.

Overview

AutoPixel runs only locally on your computer. It does not communicate with any external server and does not collect or transmit user data. The program only automates mouse and keyboard actions on screen.

1. Data We Process

AutoPixel does not collect personal data. For its operation it uses only the mouse cursor coordinates captured during calibration and the settings you enter (count, delay, jitter), held in memory while running. These are used solely to calculate click positions and are never stored or transmitted.

2. Comprehensive Disclosure

2-a. How We Collect, Use, and Store

  • Collection: No personal data is collected.
  • Use: Calibration coordinates and settings are used only to compute automated click positions.
  • Storage: Nothing is written to disk; all values are discarded when the program closes.
  • Logs / Analytics / Cookies: None.

2-b. Parties With Whom Data Is Shared

  • No data is shared with any third party.
  • No tracking, ads, cloud upload, or telemetry of any kind.

3. How It Works (System Access)

  • AutoPixel simulates real mouse movement/clicks and key presses (e.g., i) to automate wplace.
  • It does not access the file system, network, browser data, or personal information.
  • The hotkeys (F2/F3/F7/F4/Esc/F6) are detected globally only while the program is running.
  • It requires the AutoHotkey v2 runtime; AutoHotkey's behavior is governed by that software's own policies.

Additional Information

  • Security: All processing happens locally; no data ever leaves your machine.
  • Transparency: It runs as a single AutoHotkey script (AutoPixel.ahk) with fully open source.
  • AutoHotkey v2 (required runtime): AutoPixel runs on the AutoHotkey v2 runtime. Download and install it from the official site — autohotkey.com — before use. AutoHotkey is third-party software governed by its own policies.
  • Changes: If this policy changes, the date at the top will be updated.
AutoPixel is an open-source tool for personal and educational use. For inquiries, please use the GitHub repository Issues. You are responsible for your use of this automation tool.

AutoPixel — wplace 자동 픽셀

최종 업데이트: 2026-06-14 · v1.0.0

AutoPixel은 wplace.live에서 픽셀을 자동으로 그려주는 Windows용 오토마우스(AutoHotkey 스크립트)입니다. 이 문서는 AutoPixel이 사용자 정보를 어떻게 다루는지 설명합니다.

요약 — AutoPixel은 어떤 개인정보도 수집·저장·전송하지 않으며, 네트워크 통신 없이 전적으로 사용자 PC에서만 동작합니다.

개요 (Overview)

AutoPixel은 사용자의 컴퓨터에서 로컬로만 실행됩니다. 외부 서버와 통신하지 않고, 사용자 데이터를 수집하거나 전송하지 않습니다. 프로그램은 오직 화면 위의 마우스·키보드 동작을 자동화하는 역할만 합니다.

1. 처리하는 정보 (Data We Process)

AutoPixel은 개인정보를 수집하지 않습니다. 동작에 필요한 최소 범위로, 보정 시점의 마우스 커서 좌표와 사용자가 입력한 설정값(칸 수·딜레이·지터)을 프로그램 실행 중 메모리에서만 사용합니다. 이 값들은 클릭 위치를 계산하는 데만 쓰이며, 어디에도 저장되거나 전송되지 않습니다.

2. 종합 고지 (Comprehensive Disclosure)

2-a. 수집 · 이용 · 저장 방식

  • 수집: 수집하는 개인정보가 없습니다.
  • 이용: 보정 좌표·설정값은 자동 클릭 위치를 계산하는 용도로만 사용됩니다.
  • 저장: 디스크에 저장하지 않습니다. 프로그램을 종료하면 모든 값이 사라집니다.
  • 로그 · 분석 · 쿠키: 사용하지 않습니다.

2-b. 정보를 공유하는 대상 (Parties)

  • 제3자와 공유하는 데이터가 없습니다.
  • 추적(tracking), 광고, 클라우드 업로드, 텔레메트리를 일절 사용하지 않습니다.

3. 동작 방식 및 시스템 접근 (How It Works)

  • AutoPixel은 실제 마우스 이동·클릭과 키 입력(i 등)을 시뮬레이션하여 wplace 조작을 자동화합니다.
  • 파일 시스템·네트워크·브라우저 데이터·개인정보에 접근하지 않습니다.
  • 단축키(F2/F3/F7/F4/Esc/F6)는 프로그램이 실행 중일 때만 전역으로 감지됩니다.
  • 실행에는 AutoHotkey v2 런타임이 필요하며, AutoHotkey의 동작은 해당 소프트웨어의 정책을 따릅니다.

추가 정보 (Additional Information)

  • 보안: 모든 처리는 로컬에서만 이루어지며, 외부로 나가는 데이터가 없습니다.
  • 투명성: 단일 AutoHotkey 스크립트(AutoPixel.ahk)로 동작하며, 전체 소스가 공개되어 있습니다.
  • AutoHotkey v2 (필수 런타임): AutoPixel은 AutoHotkey v2 런타임에서 동작합니다. 사용 전 공식 사이트 — autohotkey.com — 에서 내려받아 설치하세요. AutoHotkey는 제3자 소프트웨어로, 해당 소프트웨어의 정책을 따릅니다.
  • 변경: 정책이 변경되면 이 문서 상단의 날짜가 갱신됩니다.
